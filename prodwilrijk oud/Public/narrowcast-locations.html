How to Use
Display URLs

Each display should open:

/narrowcast-display.html?location=location_id
Location ID

The ID is used in URLs. Use lowercase, no spaces (e.g., "productie", "kantine", "magazijn").

Display Name

The friendly name shown in admin panel (e.g., "Productie Vloer", "Kantine").

Reserved ID

The ID all is reserved and means "show everywhere".
